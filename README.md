# 🌤️ Interactive Weather Dashboard (Live Demo)

A premium, glassmorphism-styled weather application that provides real-time meteorological data with a sleek, high-end user interface. This project showcases advanced API integration, real-time data handling, and modern CSS styling.

## 🚀 Key Features

- Live Weather Updates: Fetches real-time temperature, humidity, and wind speed using the OpenWeatherMap API.
- Dynamic City Search: Allows users to search for the current weather in any city worldwide instantly.
- World Capitals Sidebar: A dedicated quick-access panel showing current conditions for major global hubs (London, Paris, Tokyo, etc.).
- Advanced Metrics: Displays detailed weather data including air pressure, visibility range, and precise Sunrise/Sunset times.
- Integrated Digital Clock: A live, real-time clock for enhanced dashboard utility.
- Premium Glassmorphism UI: Features a modern "frosted glass" aesthetic with blurred backgrounds, high-contrast typography, and responsive layouts.

## 🛠️ Tech Stack

- Frontend: HTML5, CSS3 (Flexbox, Grid, Glassmorphism techniques)
- Logic: JavaScript (ES6+, Fetch API for asynchronous data loading)
- Data Source: [OpenWeatherMap API](https://openweathermap.org/)
